# 🍽️ Aroma y Tierra 🍖

App de pedidos online para restaurante de comida criolla en Ica, Perú.

### Características:
- 40 platos criollos con precios Salón y Delivery
- Carrito de compras
- Envío de pedidos directo a WhatsApp
- Filtros por categoría y precio

### Creado con:
R + Shiny + shinydashboard
